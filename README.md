## Profile

**Mobile engineer based in Lahore, Pakistan, focused on turning product ideas into shipped, revenue-generating apps.**

I own the full lifecycle — product scoping, UI design, Flutter development, backend architecture, payments, AI integrations, and App Store / Play Store deployment — for founders who need to move from idea to production without assembling a five-person team first. Six apps I built solo are live today across both major app stores with 11,000+ combined downloads.

I don't hand off between design and engineering. I make the calls, ship the build, and iterate on real user feedback. That's the value I bring to a team: fewer handoffs, faster decisions, production-quality output.

**What I optimize for on every project:**

- Ship the first version fast, without cutting corners that cost you later
- Architect for scale so growth doesn't force a rewrite
- Keep Firebase and infra costs predictable as usage grows
- Deliver interfaces that feel considered, not templated
