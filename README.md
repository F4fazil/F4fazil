# Fazil Qazi

Mobile developer based in Lahore. Four years building Flutter apps that ship — not prototypes, not demos, live products on the App Store and Play Store.

I handle both sides: the UI decisions that most developers leave open, and the backend work that holds everything together. Firebase, Node.js, REST APIs — I connect the full stack so you are not coordinating between two people.
